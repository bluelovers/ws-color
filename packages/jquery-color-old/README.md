# jquery-color-old

jQuery Color Animations 舊版本封裝包 (Legacy jQuery Color Animations wrapper package)

## 安裝 (Installation)

```bash
# 使用 yarn / Using yarn
yarn add jquery-color-old

# 使用 yarn-tool / Using yarn-tool
yarn-tool add jquery-color-old
# yt 是 yarn-tool 的別名 / yt is an alias for yarn-tool
yt add jquery-color-old

# 使用 pnpm / Using pnpm
pnpm add jquery-color-old

# 使用 npm / Using npm
npm install jquery-color-old
```
