# @lazy-color/palette-apexcharts

ApexCharts 官方主題的預定義顏色調色板集合 (Pre-defined color palette collection from ApexCharts official themes)

## 安裝 (Installation)

```bash
# 使用 yarn / Using yarn
yarn add @lazy-color/palette-apexcharts

# 使用 yarn-tool / Using yarn-tool
yarn-tool add @lazy-color/palette-apexcharts
# yt 是 yarn-tool 的別名 / yt is an alias for yarn-tool
yt add @lazy-color/palette-apexcharts

# 使用 pnpm / Using pnpm
pnpm add @lazy-color/palette-apexcharts

# 使用 npm / Using npm
npm install @lazy-color/palette-apexcharts
```
