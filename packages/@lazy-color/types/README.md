# @lazy-color/types

顏色資料結構的 TypeScript 類型定義集合 (TypeScript type definitions collection for color data structures)

## 安裝 (Installation)

```bash
# 使用 yarn / Using yarn
yarn add @lazy-color/types

# 使用 yarn-tool / Using yarn-tool
yarn-tool add @lazy-color/types
# yt 是 yarn-tool 的別名 / yt is an alias for yarn-tool
yt add @lazy-color/types

# 使用 pnpm / Using pnpm
pnpm add @lazy-color/types

# 使用 npm / Using npm
npm install @lazy-color/types
```
