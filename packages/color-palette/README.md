# @bluelovers/color-palette

色彩調色盤集合，包含 ColorBrewer 和 W3C X11 色彩標準 (Color palette collection including ColorBrewer and W3C X11 color standards)

## 安裝 (Installation)

```bash
# 使用 yarn / Using yarn
yarn add @bluelovers/color-palette

# 使用 yarn-tool / Using yarn-tool
yarn-tool add @bluelovers/color-palette
# yt 是 yarn-tool 的別名 / yt is an alias for yarn-tool
yt add @bluelovers/color-palette

# 使用 pnpm / Using pnpm
pnpm add @bluelovers/color-palette

# 使用 npm / Using npm
npm install @bluelovers/color-palette
```
